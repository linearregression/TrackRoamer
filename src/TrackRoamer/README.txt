
This folder should reside two levels down the MRDS installation tree, for example:

	C:\Microsoft Robotics Dev Studio 4\projects\TrackRoamer


How to run the robot:

-- folder  C:\Projects\Robotics\Robotics\TrackRoamer\TrackRoamerServices\Config is copied in pre-build of TrackRoamerServices project to "C:\Microsoft Robotics Dev Studio 2008 R3\Config

-- Copy C:\Projects\Robotics\Robotics\TrackRoamer\run.bat to C:\Microsoft Robotics Dev Studio 2008 R3\bin\.

-- invoke MSRTS (DSS) command prompt:

    > run
